# PRD Documents for Rental Market Analyzer & Lease Advisor

This directory contains the Product Requirements Documents created during planning sessions. Files explain the project goals, technology choices, data sources, hosting approach, security decisions and a 4-week implementation plan.

Files in this folder:
- `01-overview.md` — high-level project overview and acceptance criteria.
- `02-tech-stack.md` — chosen technologies and rationale.
- `03-data-sources.md` — curated datasets and API integration strategy.
- `04-architecture-and-hosting.md` — repo layout, Docker Compose sample and hosting notes.
- `05-security-and-secrets.md` — secrets handling, rate-limiting and data protection.
- `06-weekly-plan.md` — 4-week implementation milestones and deliverables.
- `07-apis-and-backend.md` — API surface and ABR proxy details.
- `08-decision-log.md` — chronological decision log with rationale.
